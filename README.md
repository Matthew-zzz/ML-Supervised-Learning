**Цель:** освоить **линейную и логистическую регрессию**, метрики качества и способы их оценки в реальных задачах.

**Вводные данные:**  
Табличный датасет с целевой числовой переменной (например цены домов) _и_ с меткой класса (например “дорогой / недорогой”).

**Задачи:**

1. **Линейная регрессия:**
    
    - обучить модель предсказывать непрерывное значение;
        
    - реализовать _ручной_ расчет линейной регрессии через нормальные уравнения;
        
    - сравнить с реализацией через `sklearn.linear_model.LinearRegression`.
        
2. **Логистическая регрессия:**
    
    - классифицировать бинарные метки;
        
    - реализовать логистическую регрессию через градиентный спуск;
        
    - сравнить с `sklearn.linear_model.LogisticRegression`.
        
3. **Метрики качества моделей:**
    
    - для регрессии: MAE, MSE, RMSE, R²;
        
    - для классификации: accuracy, F1, ROC-AUC, confusion matrix;
        
    - обсудить, какая метрика почему подходит для какой задачи.
